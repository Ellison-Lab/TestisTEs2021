# Snakemake workflow: TestisTEs2021

This workflow generates outputs for the Ellison Lab Testes TE project.

## Get started

### prereqs

conda
bioconda in channels
cellranger >= 3.1.0
snakemake >= 5.30.1

### command

```
snakemake --use-conda --profile config/profile/{your system} -kp
```
